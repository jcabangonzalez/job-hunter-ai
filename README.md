# Job Hunter AI

AI-assisted job filtering pipeline using Python and Ollama.

## Overview

Job Hunter AI is a small automation tool that helps filter and analyze job postings using local LLMs.  
It scores jobs based on predefined criteria and optionally uses an AI model to evaluate job fit.

The goal is to speed up the job search process by automatically ranking opportunities.

## Features

- Job filtering based on configurable rules
- Scoring system for ranking opportunities
- AI analysis of job descriptions using Ollama
- Markdown report generation
- Simple Python CLI pipeline

## Tech Stack

- Python
- Ollama (local LLM)
- JSON configuration
- Markdown report generation
